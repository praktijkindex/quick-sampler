# Quick::Sampler

Composable samplers of data. Describe randomness and watch it blend.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quick-sampler'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quick-sampler
